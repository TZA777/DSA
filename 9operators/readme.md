        Operator Precednece----Similar to BODMAS----return boolean value 
        a. ()
        b. urinary operators-------postfix, prefix 
        c. arthematic operators----*  ,  /
        d. arthematic operators----+, -, 
        e. relational operators----==, !=, >, <, >=, <=
        f. logical operators-------&&(AND),  ||(OR),  !(NOT)

        Key Points:
        && and || use short-circuit evaluation:
        a. In A && B, if A is false, B is not checked.
        b. In A || B, if A is true, B is not checked.

        Useful in if conditions.
